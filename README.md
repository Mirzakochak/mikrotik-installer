# 🚀 نصب خودکار میکروتیک روی اوبونتو | Auto MikroTik CHR Installer

یک اسکریپت قدرتمند، سبک و کاملاً خودکار برای نصب **MikroTik RouterOS (CHR)** روی سرورهای مجازی (VPS) مبتنی بر لینوکس اوبونتو. این اسکریپت بدون نیاز به هیچ ابزار اضافه‌ای، صفر تا صد مراحل نصب و کانفیگ شبکه را برای شما انجام می‌دهد.

---

## ✨ ویژگی‌های برجسته (Features)

* **💻 Bashِ خالص:** بدون نیاز به پایتون یا نصب پیش‌نیازهای سنگین. ایده‌آل برای سرورهای مینیمال.
* **🔍 شناسایی هوشمند:** پیدا کردن خودکار درایو اصلی سرور (مانند `vda` ،`sda` یا `nvme`).
* **🌐 شبکه اتوماتیک (Auto-Provisioning):** استخراج هوشمندانه آی‌پی (IP) و گیت‌وی (Gateway) اوبونتو و تزریق آن به میکروتیک قبل از نصب.
* **⚡ آماده‌ی Winbox:** دسترسی مستقیم و بدون دردسر به وینباکس، دقیقاً بلافاصله پس از ریبوت سرور!

---

## 🛠 آموزش نصب (Installation)

برای نصب، کافیست وارد ترمینال سرور اوبونتوی خود شوید و دستور زیر را با دسترسی **`root`** اجرا کنید:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/Mirzakochak/mikrotik-installer/main/install.sh)
```
⏳ توجه: پس از اجرای اسکریپت، سرور به صورت خودکار ریبوت می‌شود. حدود ۱ الی ۲ دقیقه صبر کنید تا میکروتیک به طور کامل بوت شود.

🔐 اطلاعات ورود (Default Credentials)
پس از بالا آمدن سرور، می‌توانید از طریق Winbox و با آی‌پی سرور خود وارد شوید:

👤 نام کاربری (User): admin

🔑 رمز عبور (Password): (کاملاً خالی بگذارید و فقط Enter بزنید)

(نکته: در اولین ورود، میکروتیک از شما می‌خواهد که یک رمز عبور جدید و امن برای سرور خود تنظیم کنید).

⚠️ هشدار مهم (Disclaimer)
اجرای این اسکریپت تمام اطلاعات فعلی سیستم‌عامل اوبونتو را پاک کرده و هارد سرور را فرمت می‌کند. لطفاً این اسکریپت را فقط روی یک سرور مجازی (VPS) خام و تازه خریداری شده اجرا کنید.
